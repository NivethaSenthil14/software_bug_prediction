# Software Bug Prediction using Machine Learning

## 📌 Project Overview
This project predicts software defects using machine learning techniques on the JM1 dataset.

## ⚙️ Steps Involved
1. Data Cleaning & Preprocessing  
2. Handling Missing Values  
3. Feature Engineering  
4. SMOTE for balancing dataset  
5. Model Building  
6. Stacked Ensemble Learning  
7. Hyperparameter Tuning using Optuna  

## 🤖 Models Used
- Random Forest  
- XGBoost  
- Support Vector Machine (SVM)  
- Neural Network (MLP)  
- Stacking Ensemble  

## 📊 Evaluation Metrics
- Accuracy  
- F1 Score  
- ROC-AUC  
- PR-AUC  

## 📁 Dataset
JM1 NASA dataset for software defect prediction
